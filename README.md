# 🎱 로또 랩 (Lotto Lab)

역대 로또 당첨번호 이력으로 노는 **단일 파일 정적 토이앱**. 외부 라이브러리 없이 바닐라 JS + Canvas로만 동작합니다.

## 기능

- 🆕 **최신 회차** — 당첨번호·보너스·당첨금/인원
- 📊 **번호별 출현 빈도** — Canvas 막대그래프, 막대 클릭 시 상세
- 🔥 **핫/콜드 넘버** — 최다·최소 출현 Top6
- 🧮 **통계** — 합계 평균, 홀짝 비율, 연속번호 포함률 등
- 🎰 **행운 번호 생성기** — 랜덤/핫/콜드/합계균형/역대미출현 5가지 전략
- 🏆 **1등 닮은꼴 추천** — 역대 1등 조합 패턴(합계·홀짝·번호대·연속)과 닮은 조합을 점수화해 추천
- 🏅 **명예의 전당** — 최고 당첨금 / 1등 최다 배출 / 최소 당첨금 회차 랭킹
- 🎯 **분석 & 시뮬레이터** — 내 번호의 역대 적중 이력 분석 + "꾸준히 샀다면 수익은?" 시뮬레이션

## 사용

`index.html`을 브라우저로 열면 끝. 별도 빌드·서버 불필요.

## 면책

로또는 완전 무작위 추첨이라 모든 조합의 1등 확률은 동일(1/8,145,060)합니다. 모든 통계·추천은 **재미용**이며 당첨을 보장하지 않습니다 😉
